# Teacher-Assistance-System
A teacher-assistance-system for courses in ZJU  
软件需求工程 软件工程管理 课程项目

### 选型
- 前端
  - UI库: Element 1.0
  - Vue2.0 + Vuex + Vue-router
  - Webpack打包
- 后端
  - PHP + MySQL

### 目录结构
- frontend: 前端源代码
  - components: 组件目录
  - routes: 路由
  - vuex: vuex目录
    - modules: 各模块的store
  - index.js: 入口js
- backend: 后端源代码
  - 根据功能模块命名目录
